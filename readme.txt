4、空天地一体化化工智慧园区（）
研究内容：面向构建高度智能化的化工园区管理系统，实现由静态感知向动态演化预测的跨越，全面提升安全、环保与运营效率。首先搭建园区全要素、多尺度高精度三维数字模型，覆盖地上设备、地下管网、建筑物、道路及周边环境；其次设计并实现数据传输协议，保障空、天、地多源感知数据（高清视频、点云、环境参数、设备状态等）低时延、高可靠传输；再开发统一的数据融合平台，标准化数据与元数据，打通互操作并沉淀园区级共享数据湖。
技术指标：
（1）	初步掌握三维建模技术
（2）	掌握软件开发技术；
（3）	建模化工园区（包含相互关联的化工车间3-5个）；
（4）	实现1套数据传输协议，实现多源数据传输。
